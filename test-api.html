📖 Cómo Usar Este Testing

Flujo de Testing:

1. Simular Detección (Opción A: Manual)

  • Selecciona un marcador (Pez, Tortuga, Tiburón)
  • Click en el botón para enviar
  • Verás automáticamente el resultado en "Estadísticas"

2. Simular Detección (Opción B: Automático)

  • Click "▶️ Ejecutar Test"
  • Enviará automáticamente 3 modelos (uno cada 2 segundos)
  • Verás toda la progresión en los logs

3. Verificar API

  • Click "📡 Verificar API" para health check
  • Si verde ✅ → API activa y funcionando
  • Si rojo ❌ → Ejecutar: cd api && npm start

4. Ver Respuestas

  • Click "🔄 Obtener Último Modelo" para ver qué envió el Scanner
  • Click "📜 Ver Historial" para ver todos los modelos

5. Verificar Visualizador

  • Abre el Visualizador en otra pestaña: app-visualizer/
  • Cuando envíes un modelo aquí, debería aparecer automáticamente
  • Esto verifica que el polling funciona(cada 2 segundos)
